This is a single-page web application built as part of an assignment to demonstrate proficiency in:

Using an external API to fetch and display data.
Implementing user interaction (e.g., search and watchlist functionality), designing a user-friendly interface, writing clean code.
The app is designed to be simple and ensuring a seamless experience for users.
Features: here’s what you can do with this app--

Core Features
* Search Functionality
Users can search for TV shows or movies by title. 

* Display Media Details
Each media item shows: Title, Year of release, Type (TV show or movie), Poster image (or placeholder if unavailable)

* Watchlist Feature
Users can add media to a watchlist. The watchlist persists in the browser using localStorage. Users can remove items from the watchlist.

*A PI Integration
Fetches data from Watchmode API.
